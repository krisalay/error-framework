# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, professional, and
harassment-free experience for everyone.

## Expected Behavior

-   Be respectful
-   Be professional
-   Be constructive
-   Accept feedback gracefully

## Unacceptable Behavior

-   Harassment
-   Discrimination
-   Personal attacks
-   Trolling

## Enforcement

Maintainers may remove contributions that violate this code.

## Contact

Report violations to maintainers via GitHub Issues or email.
